このチェックリストをコピーして進行状況を追跡します。

```
- [ ] Step1: Slackのスレッドの内容を取得する。
- [ ] Step2: 何の作業をしているかを特定する。
- [ ] Step3: 内容をまとめる。
```


## Step1 Slackのスレッドの内容を取得する。

以下のコマンドを利用して、スレッドの内容を取得してください。
```
slk thread <url>
```

## Step2 何の作業をしているかを特定する。

- Upgrade
- アラート対応
- リリース作業

## Step3 内容をまとめる。

Step2で特定した作業に応じたレポートを作成する。
以下の項目は含むこと。
- 現状のステータス
- 生じている問題（未解決・解決済み）
- タスク化して改善した方がいいポイント
- 新たに得られた気づき
